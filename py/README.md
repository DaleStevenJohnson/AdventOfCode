# Advent of Code - Python Solutions

## Setup

```pwsh
./dev-setup.ps1
```

## To Run

```pwsh
python -m aoc "SessionId" 2025 1
```

Args:
- SessionId (str)
- Year (int)
- Day (int)

Options:
- --test-data (str)

`SessionId` can be found from the cookies in your browser, from the AOC website.

`test-data` allows you to pass in any arbitrary puzzle input, but is mainly for passing in the small example each puzzle comes with.
